# Meanwhile — sources, one per line. Blank lines and lines starting with # are ignored.
#
# FOUR source types are supported (all free, no API key):
#   1. An http(s) URL            -> an RSS or Atom feed
#   2. arxiv:<category>          -> the arXiv API (export.arxiv.org), no key
#   3. hn:<query>                -> the Hacker News Algolia API, no key
#   4. changelog:<atom-url>      -> a release/changelog Atom feed (GitHub releases.atom)
#
# PRIMARY SOURCES ONLY. No aggregators / curated newsletters (TLDR, Last Week in
# AI, AI Weekly, …): re-summarizing those is a copyright-risk zone. This tool reads
# ONLY syndicated titles/short summaries + links and always attributes the source.
# DIGEST (news) extracts are rewritten by Gemini at refresh; see NOTICE. Dead or
# unparseable sources are skipped automatically. All URLs below were checked live.
